Java SE Runtime Environment 8 required.

Polygon file used must be in the same folder as ArtGallery.jar file. To execute, provide the polygon filename as a command-line argument.

Example: "ArtGallery.jar StSerninH.pol"


For polygon file format explanation, see: http://www.ic.unicamp.br/~cid/Problem-instances/Art-Gallery/AGPVG/index.html
